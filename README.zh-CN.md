# Okhttp3.Test

OkHttp3 网络请求库的基本使用演示项目，涵盖同步/异步 GET/POST、文件下载与上传。

## 功能

- 同步 GET / POST 请求封装
- 异步 GET / POST 请求封装，结果通过 EventBus 回传
- 文件下载，支持断点续传和实时进度（RxJava 响应式下载）
- 文件上传（MultipartBody）

## 技术栈

- Java / Android
- OkHttp3
- RxJava2
- EventBus
- ButterKnife

## 说明

这是一个学习/演示项目，适合 Android 初学者了解 OkHttp3 的核心用法，包括同步/异步请求、下载续传和文件上传。
